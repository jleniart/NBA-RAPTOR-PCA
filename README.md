# NBA-RAPTOR-PCA

The datasets used in this project were published on data.world: https://data.world/fivethirtyeight/nba-raptor-metrics

In this project, I performed Principal Components Analysis on an NBA RAPTOR dataset. All coding was done in R.

The main code can be found in the Leniart_RAPTOR_PCA.R file.
The PCA_Plot.R file is referenced as a source file. It contains code for generating visualizations of the principal components.
